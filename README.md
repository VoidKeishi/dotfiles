## Setup
- **OS**: Arch Linux
- **WM**: i3
- **Bar**: Polybar
- **Terminal**: Kitty
- **Shell**: Zsh
- **Compositor**: Compfy
- **Launcher**: Rofi
- **File Manager**: Thunar
- **Browser**: Firefox
- **Wallpaper**: Catppuccin Street & spooky spill
- **SDDM Theme**: Sugar Candy
